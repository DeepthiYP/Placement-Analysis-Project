# Placement-Analysis-Project
🎓 Smart Placement Analytics Dashboard
📌 Project Overview
The Smart Placement Analytics Dashboard is an end-to-end Data Analytics project designed to analyze student academic performance, internships, projects, aptitude scores, and placement records to identify factors influencing placement success.
The project integrates Excel, SQL, Python, Machine Learning, and Power BI to transform raw student data into actionable insights and interactive visualizations.

🎯 Objectives
Analyze student placement trends.
Identify factors affecting placement outcomes.
Predict placement chances using Machine Learning.
Visualize key insights through an interactive Power BI dashboard.
Support data-driven decision-making for placement training and skill development.

🔄 Project Workflow
Step 1: Data Collection

Student placement data was collected and stored in Excel format.

Step 2: Data Storage

The dataset was imported into SQL for structured storage and querying.

Step 3: Data Cleaning & Preprocessing

Using Python:
Handled missing values
Removed inconsistencies
Prepared features for modeling
Step 4: Machine Learning Model

Implemented Logistic Regression to predict placement outcomes.

Target Variable:

PlacementStatus (Placed / NotPlaced)

🔍 Key Insights
1. Academic Performance Matters

Students with higher CGPA demonstrate better placement outcomes.

2. Internships Improve Employability

Students with internship experience have significantly higher placement rates.

3. Projects Enhance Placement Chances

Practical project experience positively impacts placement success.

4. Placement Training is Effective

Students who attended placement training show improved placement outcomes.

5. Aptitude Scores Influence Placements

Higher aptitude test scores correlate with better placement performance.

🤖 Machine Learning
Algorithm Used

Logistic Regression

Why Logistic Regression?

Placement prediction is a binary classification problem:

Placed
NotPlaced

Logistic Regression effectively predicts the probability of placement based on student attributes.
